CONSEJOS:

- En react se recomienda un espacio antes de la barra de las self closing tabs. 

- npx create-react-app no es la unica forma de crear una App de React, pero si la mas rapita e intuitiva porque te crea una estructura con carpetas. 

- La extension JSX tambien funciona. Esta extension nos permite ser mas especifico y se sabe que el proyecto tiene codigo en lenguaje jsx.

- A partir de 2017 ya no es indispensable importar React (import React from 'react';). No es necesaria, pero se recomienda. Solo es obligatoria cuando se trabaja con Hooks.
